铁路旅客智能问答系统 — 使用说明

1. 启动 Ollama

在终端中运行：

    ollama serve

如果已经在后台运行，可以跳过这一步。

2. 安装依赖

建议使用 Python 3.10 ~ 3.12（Python 3.13 可能无法直接安装 faiss-cpu）。

    pip install -r requirements.txt

如果提示找不到 faiss-cpu，通常是 pip 镜像源没有该包，可以改用官方 PyPI：

    pip install -r requirements.txt -i https://pypi.org/simple

3. 启动 Web 问答页面

    python src/app.py

浏览器打开 http://127.0.0.1:5000

4. 运行评估

默认使用 75 条评估集，对比三种模式（纯 LLM、向量 RAG、混合 RAG）：

    python evaluate_rag.py

脚本会输出三个文件：
- evaluation_results.csv — 逐题结果
- evaluation_summary.csv — 按题型分组的汇总
- evaluation_report.html — 可视化报告

快速调试（前 3 题）：

    python evaluate_rag.py --limit 3

5. 项目结构

核心文件在根目录，Web 服务在 src/ 目录，构建脚本在 scripts/ 目录，评估结果在 eval_results/。
